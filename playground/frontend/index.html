🚀 Como usar o Playground

1. Preparar o ambiente

🍎 macOS:
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --remote-debugging-address=0.0.0.0 --disable-web-security --disable-features=VizDisplayCompositor
🐧 Linux:
google-chrome --remote-debugging-port=9222 --remote-debugging-address=0.0.0.0 --disable-web-security
🪟 Windows:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --remote-debugging-address=0.0.0.0 --disable-web-security

Execute este comando no terminal do seu host (fora do devcontainer).

⚠️ IMPORTANTE: A flag --remote-debugging-address=0.0.0.0 é ESSENCIAL para permitir conexões do devcontainer. Sem ela, o playground não conseguirá se conectar ao Chrome!

2. Verificar conexão

Clique em "🔍 Verificar Chrome" para confirmar que o playground consegue se conectar ao Chrome.

3. Configurar sessão

Preencha os campos de configuração ou use um template rápido.

4. Executar

Clique em "🚀 Executar Sessão" e veja o Chrome abrir automaticamente no seu desktop!

💡 Dicas

  • Use Slow Motion para ver as ações acontecendo
  • Configurações de sessão são salvas automaticamente
  • Logs em tempo real mostram o progresso
  • Gere código para usar em seus projetos
